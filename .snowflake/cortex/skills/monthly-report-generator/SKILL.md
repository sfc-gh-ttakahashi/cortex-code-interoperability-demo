# 月次売上レポート生成

指定された年月の売上レポート SQL を生成・実行してください。

## レポート項目
1. 月間売上合計・注文件数・平均注文額
2. カテゴリ別売上ランキング
3. 前月比の増減率（前月データがある場合）
4. 都道府県別の注文件数

## SQL の書き方ルール
- CTE（WITH句）を使って段階的に組み立てる
- QUALIFY + ROW_NUMBER でランキングを算出する（Snowflake 最適化）
- 金額は NUMBER(10,2) でフォーマットし、通貨記号は付けない
- 結果には SQL コメントで各 CTE の目的を記述する

## 出力
- 生成した SQL とその実行結果を表示する
- 結果のサマリーを日本語で1-2文で添える
